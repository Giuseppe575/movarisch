Technical documentation

This page gathers the key operational information required to run the MOVARISCH SDS parser during chemical risk assessments.

Requirements

  • Modern browser (Chrome, Edge or Firefox) with JavaScript enabled.
  • SDS/TDS files in PDF format without password protection.
  • Word export needs an internet connection to load the docx library.

Workflow

  1. Upload one or more PDFs with the “Select PDF” button.
  2. Run “Extract & Calculate” to collect H-codes and pre-fill the MOVARISCH table.
  3. Complete missing data (quantity, exposure time, distance) before finalising.
  4. Export the report in Excel or Word for your internal archive.

Quality checks

The “Test library” button is executed after each release to ensure the H-code extraction engine still works. The bundled sample PDF includes H317 and H335 for validation.

Contact

Report bugs or feature requests via support@movarisch.local. Please mention the customer and attach the reference PDF.
